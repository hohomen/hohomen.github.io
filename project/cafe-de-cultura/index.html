cafe de cultura



Cafe De Cultura는 스프링 학습을 목적으로 진행하고 있는 개인 프로젝트입니다.

기획
카페에서 우리는 영화, 연애, 여행 등의 다양한 주제를 가지고 이야기를 합니다.
Cafe De Cultura는 특히 문화를 주제로 한 온라인 소통 공간입니다. 문화에 관심있는 사람이라면 누구나 이곳에서 책, 영화, 공연에 대해서 이야기를 주고 받으며 소통할 할 수 있습니다.




(메인 화면)






개발환경
언어
HTML5,  CSS,  JavaScript(jQuery),  jAVA8
사용 기술
Bootstrap, Spring(4.0), MyBatis
도구
Eclipse(Neon),  MY-SQL(5.7)
서버
AWS(Ubuntu Linux),  Tomcat8(WAS)

구현 기능

  • 게시판 기능(CRUD)

학습 내용

  • DI/AOP
  • Maven 디렉토리 구조와 Spring 환경설정(XML_Files & Path)
  • Spring MVC 디자인 패턴을 통한 개발(MyBatis)

프로젝트 보완 방향

  • 게시판 기능 보완
  • 로그인 처리와 스프링 시큐리티
  • Rest 방식의 개발
  • 트랜잭션
  • 웹 소켓 활용
