MVC Controller

March 20, 2019     ""

2 minute read


스프링 MVC를 이용한 웹 개발의 핵심인 컨트롤러의 구현 방법을 살펴보자.



컨트롤러 구현 과정은 다음과 같다.
1. @Controller 애노테이션을 클래스에 적용한다.
2. @ReaquestMapping 애노테이션을 이용해서 처리한 요청 경로를 지정한다.
3. 웹 브라우저의 요청을 처리할 메서드를 구현하고, 뷰 이름을 리턴한다.



메서드는 뷰이름을 리턴하기 전에 다음의 두 가지 작업을 수행한다.
- 클라이언트의 요청을 처리한다.
- 처리 결과 데이터를 뷰에 전달한다.

다양한 방식으로 모델에 데이터를 담을 수 있는데
가장 간단한 방법은 Model을 파라미터로 추가하고, Model 파라미터에 데이터를 추가하는 것이다.
Model 클래스는 addAttribute( ) 메서드를 제공하고 있다.
