XML 특수 기호

February 17, 2019     ""

2 minute read

xml 문서에서 특수 기호를 사용하려면 개체 참조(Entity Reference)나 CDATA(Character Data)를 사용해야 한다.




1, 개체 참조
개체 참조는 미리 정해진 특별한 코드나 이름을 사용할 때 쓰이는 개념으로, 개체에 할당된 이름이 개체명이다. 예를 들면 ,< 기호를 표현하는 ‘lt’라는 개체명은 ‘less than’의 머리 글자로 정의한 것이다. 개체 참조는 &와 ;을 사용하여 나타내며, 개체 참조로 특수 기호를 표현할 수 있다.



ex)
표현 문자 <     개체 참조명 &lt;    어원 less then
표현 문자 >     개체 참조명 &gt;    어원 greater then
표현 문자 ‘     개체 참조명 &apos;  어원 apostrophe
표현 문자 “     개체 참조명 &quot;  어원 quotation marks
표현 문자 &     개체 참조명 &amp;   어원 ampersand
2. CDATA 특수 기호를 많이 사용하는 경우에는 CDATA 섹션을 사용하여 쉽게 표현할 수 있다. CDATA 섹션이라는 것은 문자 데이터를 XML 데이터로 해석하지 않고 그대로 표현하는 것을 말한다.

기본 형식)

<![CDATA[
    
    ]]>

예제)

    <student>
        <sid> 100 </sid>
        <name> 홍 현 </name>
        <activity>
            <![CDATA[ 1992년 "단국대" 입학. 
                1993년 학생회장 & 과대표. 
                1996년 "단국대" 졸업. 
                2000년 << WG 입사 >>
                2003년 << 총동창 회장 & 국회의원 >>
            ]]>
        </activity>
    </student>

< 나 &와 같은 기호에 대한 개체 참조명이나 문자 참조를 사용하지 않아도 제대로 표현되는 것을 확인할 수 있다.

CDATA를 사용하여 XML 문서를 작성할 때 잘못된 표현 예
1. <![CDATA[  ]]>에 공백을 쓸 수 없다.
ex) <![(공백)CDATA[ ]]>
2. CDATA섹션 안에 CDATA 섹션을 포함하지 않아야 한다.
3. 키워드 CDATA는 반드시 대문자로 쓴다.
4. CDATA 구역은 요소 콘텐츠 내의 문자 데이터 어디에나 삽입할 수 있다.(단, XML 마크업(태그 영역)내에서는 사용할 수 없다.
ex)

    <student>
        <sid> 100 </sid>
        <name> 홍 현 </name>
        <activity <![CDATA[ 1992년 "단국대" 입학. ]]></activity>
    </student>

참고 문헌: XML 프로그래밍(홍성용, 한빛아카데미)
