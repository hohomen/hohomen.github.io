CSS 기초

February 18, 2019     ""

2 minute read

HTML5 + CSS3 웹 표준의 정석의 내용을 요약한 것입니다.

1. 스타일과 스타일 시트

스타일 
웹 문서에서 스타일(style)이란 HTML 문서에서 자주 사용하는 글꼴이나 색상, 정렬, 각 요소들의 배치 방법 등 
문서의 겉모습을 결정짓는 내용들을 가리킵니다. 웹 문서의 내용과 상관없이 디자인만 바꿀 수 있습니다.

스타일 형식 p(=선택자) { text-align(=스타일 속성): center(=속성 값) }

스타일 주석 /* */

스타일 시트(내부 스타일 시트와 외부 스타일 시트로 나뉜다.)

내부 스타일 시트 
<style></style> 안에 정의

외부 스타일 시트 
<link href="외부 스타일 파일 경로" rel=stylesheet" type="text/css">  

인라인 스타일(간단한 스타일 정보라면 스타일 시트를 사용하지 않고 스타일을 적용할 대상에 직접 표시합니다.)
<p style="color:blue;"> 블루베리는 항산화제인 ....... </p>


2.선택자

태그 선택자
p{
font-size: 12px;
font-family: 돋움;
}

클래스 선택자
같은 태그라도 어떤 곳에서는 다른 스타일을 사용하고 싶을 때
특정 부분에만 스타일을 적용할 때 사용하는 것이 클래스 선택자.
.클래스명 { 스타일 }
.bluetext {
color:blue;
}
<h2 class="bluetext">블루베리와 항산화 효능</h2>

만약 다른 태그에 클래스 선택자를 적용하고 싶지 않다면 클래스 선택자 앞에 태그를 써서 해당 태그에만 클래스
스타일을 적용할 수 있다.
h2.bluetext {
coler:blue;
}

요소 전체가 아니라 일부에만 클래스 선택자를 적용하고 싶다면 <span>태그를 사용하면 된다.
<span class="redtext">안녕하세요 </span>

아이디 선택자
클래스 선택자와 마찬가지로 웹 문서 안의 특정 부분에 스타일을 지정할 때 사용한다. 마침표(.) 대신 (#)기호를
사용한다는 점과 id 선택자는 클래스 선택자와 달리 문서 안에서 한 번만 적용할 수 있다는 차이가 있다.(div태그
와 같이 많이 사용,)

#아이디명 { 스타일 }

그룹 선택자 
여러 선택자에 같은 스타일이 사용되는 경우에 쉽표(,)로 구분해 여러 선택자를 나열한 후 스타일은 한 번만 정의.
h1, h2 {
text-align:center;
}


3.캐스캐이딩 스타일 시트(CSS)
스타일 시트는 사실 CSS라는 명칭으로 더 많이 알려져 있습니다. 
CSS와 스타일 시트는 비슷해 보이지만 조금 차이가 있습니다. '스타일 시트'라는 용어 앞에 '카스캐이딩(cascad
ing)'이라는 용어가 하나 더 붙은 것이 CSS죠. 그렇다면 카스캐이딩은 무슨 의미이며 어떤 역할을 할까요?
CSS 카스캐이딩: '위에서 아래로 흐르는 스타일 시트' 
'카스캐이딩'은 선택자에 적용된 많은 스타일 중에 어떤 스타일을 나타낼지를 결정함을 뜻합니다.

우선순위는 다음의 세가지 개념에 따라 지정됩니다. 
1.얼마나 중요한가 - 중요도(Importance) 2.얼마나 한정지을 수 있는가 - 명시도(Specificity) 3.소스에서의
순서(Source Order)

중요도(*p.215 그림)
a. 사용자 스타일 시트가 최우선!
b. 중요 스타일
c. 기본적인 브라우저 스타일 시트

명시도(p.217)
