Javascript 기본 문법

February 17, 2019     ""

3 minute read

‘모던 웹을 위한 JavaScript jQuery입문 3판’의 내용을 요약한 것입니다.


CHAPTER 2 기본 문법 
2.1 기본 용어 
2 키워드
자바스크립트가 처음 만들어질 때 정해진 특별한 의미가 있는 단어를 '키워드'라고 합니다. (사진P.28)

3 식별자
식별자를 만들 때는 다음의 규칙을 지켜야 합니다.
1.키워드를 사용하면 안 됩니다.
2.숫자로 시작하면 안 됩니다.
3.특수 문자는 _와 $만 허용합니다.
4.공백 문자를 포함할 수 없습니다.
이 네 가지 규칙만 만족한다면 모든 단어를 식별자로 사용할 수 있습니다

식별자를 만들 때 알파벳을 사용하는 것 외에도 자바스크립트 개발자 대부분이 지키는 관례가 있습니다.
1. 생성자 함수의 이름은 항상 대문자로 시작합니다.
2. 변수와 인스턴스, 함수, 메서드의 이름은 항상 소문자로 시작합니다.
3. 여러 단어로 이루어진 식별자는 각 단어의 첫 글자를 대문자로 합니다.

자바스크립트의 식별자는 크게 네 종류로 나눌 수 있습니다.(보편적으로)
함수 alert('Hello World')
속성 Array.length 
변수 input
메서드 Math.abs(-273)

2.2 출력 
자바 스크립트의 가장 기본적인 출력 방법은 alert() 함수를 사용하는 것입니다.
함수의 괄호 안에는 문자열을 입력합니다.

2.3 문자열 자료형 
자바스크립트는 두 가지 방법으로 문자열을 생성합니다.
큰 따옴표와 작은 따옴표를 사용합니다.

내부에 작은따옴표를 사용하고 싶다면 외부에 큰따옴표를 사용합니다. 반대로 내부에 큰따옴표를 사용하고 싶으면 외부에 작은따옴표를 사용합니다.
한 가지 따옴표로 일관되게 쓰고 싶다면 특수한 기능을 수행하는 이스케이프 문자(\)를 사용합니다.
이스케이프 문자는 이외에도 여러 가지 특수 기능이 있습니다. \n과 같은.

2.4 숫자 자료형 
정수와 유리수의 구분 없이 모두 같은 자료형으로 인식합니다.

2.5 불 자료형 
자바 스크립트에는 다음과 같은 비교 연산자가 있습니다.
>=  <=  >   <   ==  !=
alert(52 > 173); --> false 
alert(52 < 173); --> true

논리 연산자 
  ! &&  ||
alert(30 > 20 > 10); --> false
(= alert 1 > 10); 과 같음

2.6 변수
변수는 값을 저장하는 식별자입니다. 변수는 숫자 자료형 뿐만 아니라 모든 _자료형_을 저장할 수 있습니다.
자바스크립트에는 문자열, 숫자, 불, 함수, 객체, undefined 총 여섯 가지 자료형이 있습니다.
변수에 저장된 자료형에 따라 지금까지 배운 모든 연산이 가능합니다.
var stringVar = 'String';
var numberVar = 237;
var booleanVar =true;
var functionVar = function () { };
var objectVar = { };

변수는 복합 대입 연산자와 증감 연산자를 사용할 수 있습니다.
복합 대입 연산자는 대입 연산자와 다른 연산자를 함께 사용하는 연산자입니다.
+=  -=  *=  /=  %=
증감 연산자는 복합 대입 연산자를 약간 간략하게 사용한 형태입니다.
Variable++  ++Variable  Variable--      --Variable
 
2.7 자료형 검사.
typeof 연산자를 사용합니다.
alert(typeof ('String')); //문자열
alert(typeof (273)); // 숫자
alert(typeof (true)); //불
alert(typeof (function () { })); // 함수
alert(typeof ({})); //객체
alert(typeof (alpha)); //undefined 
2.8 undefined 자료형 

2.9 입력 
이 절에서는 문자열 자료형과 불 자료형을 입력하는 방법을 살펴봅시다.
문자열 var input = prompt('Message', 'DefStr');
불  var input = confirm('수락하시겠습니까?');

2.10 숫자와 문자열 자료형 변환 
자바스크립트는 자동으로 자료형을 변환하는 경우가 많습니다.
alert('52'+ 273); //52273
alert('52' * 273); //14196
더하기 연산자를 제외한 사칙 연산자는 문자열 자료형을 숫자 자료형으로 자동 변환합니다.

강제로 자료형을 변환
Number( ) (숫자가 아닐 때는 NaN값을 출력합니다, 자바스크립트는 복소수를 표현할 수 없습니다. 루트-3과 같은 복소수는 NaN으로 표시됩니다.)
String( ) 

2.11 불 자료형 변환 
불 자료형으로 변환할 때는 Boolean( ) 함수를 사용합니다.
다섯가지를 제외한 모든 경우에 true로 변환됩니다.
alert(Boolean(0));  alert(Boolean('')); alert(Boolean(NaN));    alert(Boolean(null));       **alert(Boolean(undefined));**

2.12 일치 연산자 
비교 연산자를 사용할 때 자동으로 자료형이 변환되어 뜻하지 않는 경우가 발생할 수 있습니다.
alert('' == false); //true
alert('' == 0); //true
alert(0 == false); //true
alert('273' == 273); //true

이러한 유연성으로 인해 원하는 결과가 나오지 않게 하기 위해 
일치 연산자 ===, !==를 사용하면 false의 결과가 나온다.
