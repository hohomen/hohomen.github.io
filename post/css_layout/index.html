2 minute read

CSS와 박스 모델


박스 모델은
'블록 레벨(block-level)' 요소인지 '인라인 레벨(Inline-level)' 요소인지에 따라 나열 방법이 달라집니다.

블록 레벨(block-level) 요소
태그를 사용해 요소를 사입했을 때 혼자 한 줄을 차지하는 요소입니다. 너비나 마진, 패딩 등을 이용해 크기나 위치를 지정하려면 블록 레벨 요소여야 합니다. <div> 태그나 <p> 태그 등
블록 레벨 태그 p, ,h1~h6, ul, ol, pre, dl, div, noscript, blockquote, form, hr, table, fieldset, address, article

인라인 레벨(Inline-level) 요소
줄을 차지하지 않는 요소입니다. 나머지 공간에는 다른 요소가 올 수 있습니다. <img> 태그나 <strong> 태그 등
인라인 레벨 태그 img, object ,br, sub, sup, span, input, textarea, label, button

웹 문서 안에서 여러 요소들을 원하는 위치에 배치하려면 CSS 박스 모델에 대해 잘 알고 있어야 합니다.
박스 모델(은 4가지 요소로 구성됩니다.)
마진(margin)  <-  | 테두리(border) |  -> 패딩(padding) | 콘텐츠 영역
상하좌우 네 방향으로 나뉘어져 있어 네 방향의 스타일을 따로 설정할 수 있습니다.
(개발자 도구 ctrl + shift + i )



width, height 속성 - 콘텐츠 영역의 크기
속성: <크기>, <백분율>, <auto>(높이와 너비 값이 콘텐츠 양에 따라 자동으로 결정됩니다. 기본 값입니다.)

실제 콘텐츠 크기 계산하기(모던 브라우저)(익스플로러 6는 다름)
모던 브라우저에서 박스 모델의 넓이 = width 값 + 좌우 패딩 + 좌우 테두리
.bx1{
    width:200px;
    height:100px;
    background: #ff6a00;
    padding:10px;
    border: 5px #ccc solid;
}


display 속성 - 화면 배치 방법 결정하기(p.311)
display 속성을 사용하면 블록 레벨 요소를 인라인 레벨 요소로 바꾸거나 인라인 레벨 요소를 블록 레벨 요소로 바꿀 수 있습니다. --> 세로로 표시되는 목록을 가로 내비게이션으로 바굴 때, 한 줄로 표시되는 이미지에 여백과 테두리를 추가해 갤러리로 표시할 때 이 방법을 사용합니다.
display : none | block | inline | inline-block | table | table-cell

inline-block 속성: display: inline; 으로 지정하면 한 줄로 배치할 수는 있지만 너비나 높이, 마진 같은 박스 모델 값이 정확히 적용되지 않습니다. 요소를 인라인 레벨로 배치하면서 내용에는 블록 레벨 속성을 지정하고 싶을 때 사용합니다.
none 속성: 이 속성 값을 지정하면 해당 요소를 화면에 아예 표시하지 않습니다. visibility: hidden; 도 비슷한 역할을 하는데 visibility 속성은 화면에서 감추기만 할 뿐 원래 요소가 있는 공간은 그대로 차지하지만 display: none;은 아예 공간조차 차지하지 않습니다.
